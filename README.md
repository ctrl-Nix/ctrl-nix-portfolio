## 🚀 Features

- **🎮 Global XP Progression System**: Earn Hacker XP (HXP) and level up as you explore the hidden corners of the portfolio!
- **🎧 Immersive Audio Engine**: Reactive synthesis audio feedback on clicks, hovers, level-ups, and CTF challenges.
- **⚡ Cinematic Transitions & Effects**: Experience screen shakes, chromatic glitches, and hyper-warp page routing when bypassing system "security".
- **🎯 Custom Target-Lock Cursor**: Responsive tracking crosshairs for UI precision seamlessly woven into the canvas background.
- **🚩 Embedded CTF Mini-game**: Hunt for embedded flags throughout the source code, decrypt Caesar ciphers, decode Base64, and prove your mettle.
- **💻 Simulated Web Terminal**: A fully functional command-line interface directly integrated into the DOM for executing commands like `whois`, `binary`, `hex`, and `decode`.
